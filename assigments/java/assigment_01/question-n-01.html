1 Question N° 01

Create an executable class called “CircleArea”, which calculates the area of a circle:

  • The radius of the circle is 4 units

I implemented and set the variables circle_radius and circle_area in the constructor of the class. After that, I created a method CircleArea.print to display the results.

/** Executable class to calculate the area of specific circle*/ 
public class CircleArea {
    Double circle_radius;
    Double circle_area;

    // Create the constructor
    public CircleArea(Double radius) {   
        circle_radius = radius;
        circle_area = Math.PI*radius*radius;
    }
    
    // Print results in a pretty way!
    public void print () {              
        System.out.format(
            String.format(
                "The area of the circle is: %.4f units\n",
                 this.circle_area
            )
        );
    }
    
    public static void main(String[] args) {
        CircleArea circle = new CircleArea(4.0);
        circle.print();
    }
}

Results:

## The area of the circle is: 50.2655 units
